The Disk Drive is a peripheral device available for CraftOS. Type "help peripheral" to learn about using the Peripheral API to connect with peripherals. When a Disk Drive is connected, peripheral.getType() will return "drive".

Methods exposed by the Disk Drive:
isDiskPresent()
getDiskLabel()
setDiskLabel( label )
hasData()
getMountPath()
hasAudio()
getAudioTitle()
playAudio()
stopAudio()
ejectDisk()
getDiskID()

Events fired by the Disk Drive:
"disk" when a disk or other item is inserted into the drive. Argument is the name of the drive.
"disk_eject" when a disk is removed from a  drive. Argument is the name of the drive.
Type "help events" to learn about the event system.

Дисковод — это периферийное устройство, доступное для CraftOS. Введите «help peripheral», чтобы узнать об использовании периферийного API для подключения к периферийным устройствам. Когда дисковод подключен, peripheral.getType() вернет «drive».

Методы, предоставляемые дисководом:
isDiskPresent()
getDiskLabel()
setDiskLabel( label )
hasData()
getMountPath()
hasAudio()
getAudioTitle()
playAudio()
stopAudio()
ejectDisk()
getDiskID()

События, запускаемые дисководом:
«disk», когда диск или другой элемент вставлен в дисковод. Аргумент - имя диска.
«disk_eject» при извлечении диска из дисковода. Аргумент - имя диска.
Введите «help events», чтобы узнать о системе событий.
