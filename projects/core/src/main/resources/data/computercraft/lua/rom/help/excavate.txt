excavate is a program for Mining Turtles. When excavate is run, the turtle will mine a rectangular shaft into the ground, collecting blocks as it goes, and return to the surface once bedrock is hit.

ex:
"excavate 3" will mine a 3x3 shaft.

excavate — это программа для майнинг-черепах. При запуске раскопок черепаха будет копать прямоугольную шахту в землю, собирая блоки по мере продвижения, и вернется на поверхность, как только ударится о коренную породу.

Использование:
«excavate 3» будет добывать шахту 3x3.
