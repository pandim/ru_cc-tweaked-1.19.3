The function os.pullEvent() will yield the program until a system event occurs. The first return value is the event name, followed by any arguments.

Some events which can occur are:
"char" when text is typed on the keyboard. Argument is the character typed.
"key" when a key is pressed on the keyboard. Arguments are the keycode and whether the key is a repeat. Compare the keycode to the values in keys API to see which key was pressed.
"key_up" when a key is released on the keyboard. Argument is the numerical keycode. Compare to the values in keys API to see which key was released.
"paste" when text is pasted from the users keyboard. Argument is the line of text pasted.

Events only on advanced computers:
"mouse_click" when a user clicks the mouse. Arguments are button, xPos, yPos.
"mouse_drag" when a user moves the mouse when held. Arguments are button, xPos, yPos.
"mouse_up" when a user releases the mouse button. Arguments are button, xPos, yPos.
"mouse_scroll" when a user uses the scrollwheel on the mouse. Arguments are direction, xPos, yPos.

Other APIs and peripherals will emit their own events. See their respective help pages for details.

Функция os.pullEvent() будет возвращать программу до тех пор, пока не произойдет системное событие. Первое возвращаемое значение — это имя события, за которым следуют любые аргументы.

Некоторые события, которые могут произойти:
"char", когда текст набирается на клавиатуре. Аргумент — это типизированный символ.
"key" при нажатии клавиши на клавиатуре. Аргументы - это код клавиши и является ли клавиша повторением. Сравните код клавиши со значениями в API ключей, чтобы увидеть, какая клавиша была нажата.
"key_up" при отпускании клавиши на клавиатуре. Аргумент — это числовой код ключа. Сравните со значениями в API ключей, чтобы узнать, какой ключ был выпущен.
«paste», когда текст вставляется с клавиатуры пользователя. Аргумент — это вставляемая строка текста.

События только на продвинутых компьютерах:
«mouse_click», когда пользователь щелкает мышью. Аргументы: кнопка, xPos, yPos.
«mouse_drag», когда пользователь перемещает мышь, удерживая ее. Аргументы: кнопка, xPos, yPos.
«mouse_up», когда пользователь отпускает кнопку мыши. Аргументы: кнопка, xPos, yPos.
«mouse_scroll», когда пользователь использует колесо прокрутки мыши. Аргументы: направление, xPos, yPos.

Другие API и периферийные устройства будут генерировать свои собственные события. Подробности смотрите на соответствующих страницах справки.
