To learn the lua programming language, visit http://lua-users.org/wiki/TutorialDirectory.

To experiment with lua in CraftOS, run the "lua" program and start typing code.
To create programs, use "edit" to create files, then type their names in the shell to run them. If you name a program "startup" and place it in the root or on a disk drive, it will run automatically when the computer starts.

To terminate a program stuck in a loop, hold Ctrl+T for 1 second.
To quickly shutdown a computer, hold Ctrl+S for 1 second.
To quickly reboot a computer, hold Ctrl+R for 1 second.

To learn about the programming APIs availiable, type "apis" or "help apis".
If you get stuck, visit the forums at http://www.computercraft.info/ for advice and tutorials.

Чтобы изучить язык программирования lua, посетите http://lua-users.org/wiki/TutorialDirectory.

Чтобы поэкспериментировать с lua в CraftOS, запустите программу «lua» и начните вводить код.
Чтобы создавать программы, используйте «edit» для создания файлов, а затем введите их имена в оболочке, чтобы запустить их. Если вы назовете программу «аstartup» и поместите ее в корень или на диск, она будет запускаться автоматически при запуске компьютера.

Чтобы завершить программу, застрявшую в цикле, удерживайте Ctrl+T в течение 1 секунды.
Чтобы быстро выключить компьютер, удерживайте Ctrl+S в течение 1 секунды.
Чтобы быстро перезагрузить компьютер, удерживайте Ctrl+R в течение 1 секунды.

Чтобы узнать о доступных API-интерфейсах программирования, введите «apis» или «help apis».
Если вы застряли, посетите форумы по адресу http://www.computercraft.info/ для получения советов и руководств.
