cp copies a file or directory from one location to another.

ex:
"cp rom myrom" copies "rom" to "myrom".
"cp rom mystuff/rom" copies "rom" to "mystuff/rom".
"cp disk/* disk2" copies the contents of one disk to another

cp копирует файл или каталог из одного места в другое.

использование:
"cp rom myrom" копирует "rom" в "myrom".
"cp rom mystuff/rom" копирует "rom" в "mystuff/rom".
"cp disk/* disk2" копирует содержимое одного диска на другой
