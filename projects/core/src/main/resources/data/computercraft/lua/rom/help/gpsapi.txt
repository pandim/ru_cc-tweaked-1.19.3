Functions in the GPS API:
gps.locate( timeout )

The locate function will send a signal to nearby gps servers, and wait for responses before the timeout. If it receives enough responses to determine this computers position then x, y and z co-ordinates will be returned, otherwise it will return nil. If GPS hosts do not have their positions configured correctly, results will be inaccurate.

Функции в GPS API:
gps.locate( timeout )

Функция определения местоположения отправит сигнал ближайшим серверам GPS и будет ждать ответов до истечения времени ожидания. Если он получает достаточно ответов для определения положения этого компьютера, то будут возвращены координаты x, y и z, в противном случае он вернет ноль. Если хосты GPS не настроили свои позиции правильно, результаты будут неточными.
