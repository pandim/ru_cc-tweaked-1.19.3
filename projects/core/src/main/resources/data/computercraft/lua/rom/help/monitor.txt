monitor will connect to an attached Monitor peripheral, and run a program on its display.
Type "help monitors" for help using monitors as peripherals in lua programs.

ex:
"monitor left hello" will run the "hello" program on the monitor to the left of the computer.
"monitor top edit foo" will run the edit program on the top monitor, editing the file "foo".

монитор - подключиться к периферийному устройству монитор и запустит программу на его дисплее.
Введите "help monitors" для получения справки по использованию мониторов в качестве периферийных устройств в программах lua.

Использование:
«monitor left hello» запустит программу «hello» на мониторе слева от компьютера.
"monitor top edit foo" запустит программу редактирования на верхнем мониторе, отредактировав файл "foo".
